## Project

Welcome to the Feed: Concept Reinforcement and Styling Practice. This project was designed to solidify key React concepts, including:

- Immutability
- React hooks
- Declarative programming
- React rendering flow
- Styling using CSS modules
- Layout replication using Figma

>`Alert 1:` Due to some features in the styling, it is not recommended to use IE 6-8 as a browser when using this application. <br>

## Environment

Before you get started, ensure you have the following installed:

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org/en)
- [Npmjs](https://www.npmjs.com/)

### Setup Instructions:

1. Clone the repository: 
    ```bash 
    https://github.com/CaioLemec/feed.git
    ```

2. Install all dependencies by running:
    ```bash
    npm i
    ```

3. Start the application in <strong>dev</strong> mode:
    ```bash
    npm run dev
    ```

4. Access the application at the following address in your browser:
    ```bash
    http://localhost:5173/
    ```
